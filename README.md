# API Endpoints


### GET
- /api.spacexdata.com/v3/rockets (Rockets)
- /api.spacexdata.com/v3/launches/upcoming (Upcoming Launches)
- /api.spacexdata.com/v3/launches/past (Past Launches)




# Environment Set Up

* npm install
* npm run start



# Run Tests

* To run all tests, run : ```npm test```
